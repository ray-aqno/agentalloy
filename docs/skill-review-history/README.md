# Skill Review History

This directory contains per-skill QA reports generated by the AgentAlloy author-critic pipeline.

## Purpose

Each report documents the quality review of a skill against the R1-R8 quality contract defined in the skill authoring pipeline. Reports are generated automatically when a skill is authored or updated.

## File Naming Convention

```
<skill_name>-<timestamp>.md
```

Examples:

```
skill-review-history/test-driven-development-2026-05-24.md
skill-review-history/systematic-debugging-2026-05-24.md
skill-review-history/deploy-pr-2026-05-24.md
```

## Report Contents

Each report includes:

- **Skill metadata** — name, category, version
- **R1-R8 evaluation** — pass/fail for each quality criterion:
  - R1: Trigger conditions clearly stated
  - R2: Steps are numbered and actionable
  - R3: Pitfalls section present and specific
  - R4: Verification steps included
  - R5: Commands are copy-paste ready
  - R6: No aspirational content (all claims verified against actual code)
  - R7: Cross-references to related skills are accurate
  - R8: Output fits within context window constraints
- **Critic feedback** — specific issues found and recommended fixes
- **Final verdict** — approved, needs revision, or rejected

## Generation

Reports are generated by the author-critic pipeline as part of the skill authoring workflow. They are not manually written.

To generate a report for an existing skill, run the authoring pipeline with the review step enabled. See [skill-authoring-and-overrides-spec.md](../skill-authoring-and-overrides-spec.md) for pipeline details.
